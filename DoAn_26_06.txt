Công việc làm hôm nay: Đăng Nhập và Quản lý Trạng thái với AuthContext, Hiển thị Danh sách Sản phẩm trên Trang Chủ, Xây dựng Trang Chi tiết Sản phẩm, Xây dựng Chức năng Giỏ hàng (Shopping Cart)
+Tạo AuthContext để quản lý trạng thái đăng nhập
	1/Trong thư mục src/contexts/, tạo một file mới tên là AuthContext.js.
+Bọc ứng dụng bằng AuthProvider
	1/Mở file src/index.js. Import AuthProvider và bọc nó bên ngoài <App />.
+Cập nhật authService.js và xây dựng LoginPage.js
	1/Thêm hàm loginUser vào authService.js
	2/Xây dựng LoginPage.js
+Cập nhật Navbar để phản ánh trạng thái đăng nhập
	1/Cập nhật file src/components/layout/Navbar.js.
+Tạo Service để lấy dữ liệu Template
	1/Trong thư mục src/services/, tạo một file mới tên là templateService.js. templateService.js
+Tạo Component ProductCard để hiển thị sản phẩm
	1/Trong thư mục src/components/, tạo một thư mục mới tên là products.
	2/Bên trong src/components/products/, tạo file ProductCard.js. ProductCard.js
+Cập nhật HomePage.js để gọi API và hiển thị dữ liệu
	1/Mở file src/pages/HomePage.js.
+Cải tiến Backend để hỗ trợ URL thân thiện (SEO-friendly)
	1/Cập nhật TemplateRepository
	2/Cập nhật TemplateService và TemplateServiceImpl
	3/Cập nhật TemplateController
+Xây dựng Trang Chi tiết Sản phẩm ở Frontend
	1/Cập nhật templateService.js
	2/Tạo trang ProductDetailPage.js
	3/Thêm Route mới vào App.js
+Tạo CartContext để quản lý giỏ hàng
	1/Trong thư mục src/contexts/, tạo một file mới tên là CartContext.js.
+Bọc ứng dụng bằng CartProvider
	1/Mở file src/index.js.
	2/Import CartProvider và bọc nó bên trong AuthProvider.
+Tích hợp chức năng "Thêm vào giỏ hàng"
	1/Cập nhật ProductCard.js
	2/Cập nhật ProductDetailPage.js
	3/Hiển thị số lượng sản phẩm trên Navbar
+Xây dựng Trang Giỏ hàng (CartPage)
	1/Tạo Component CartItem (Tái sử dụng)
	2/Tạo Trang CartPage.js
	3/Thêm Route cho Trang Giỏ hàng
